Chores List

Here is a list of all our chores.

You can see assigned chores according to the person in charge for the week.

*Important: Make sure to add at least one roommate in order to use this functionality. Please click on the "Roommate List" tab and add a roommate by following the instruction in the page.

Usage:

a) Adding a chore
  1. 1. Select the week that you want to see by adjusting the week with the right and left arrow button.
  2. 2. Click the "+" button to add a chore by filling in the "chore" information, assigning to the main roommate to start with, with the dropped down menu in "Assigned Roommate" section, and adding a description for how to handle the chores or special instruction if needed. Click "Continue" when this section is done.
  3. 3. You should now be in the "Edit Chore" section. Click on the checkbox next to the name of other roommates who will be working on this chore. The system will by default rotate the chores weekly based on who are assigned to this chore. Click "create" button when done to create this chores card.

b) Editing a chore
  1. 1. Click on the chore card that you want to edit.
  2. 2. Edit the information and click "continue" when done.
  3. 3. Change assignees if needed. Click the "Edit" button when done. Check the rotation by clicking on the following few weeks!
  4. 4. Click on the "delete" button at any point if you want to delete this chore.
